AWS Services Overview

Sr No. Service Name Description
1Cloud-Based DeploymentA cloud-based deployment runs all parts of the application in the cloud. Resources are provisioned as needed to scale.
2On-Premises DeploymentResources are deployed on a private infrastructure, offering complete control and security for sensitive workloads.
3Hybrid DeploymentCombines cloud-based and on-premises resources for flexibility.
4Amazon EC2Provides scalable, secure virtual servers for running applications.
5Amazon ELBDistributes incoming application traffic for fault tolerance and scalability.
6Amazon S3Scalable object storage for data archiving, backup, and applications.
7Amazon RDSManaged relational database service supporting MySQL, PostgreSQL, and more.
8AWS LambdaServerless compute service that runs code in response to events.
9Amazon CloudFrontContent delivery network (CDN) for low-latency content distribution.
10Amazon Route 53Scalable DNS service for routing users to AWS-hosted applications.
11Amazon DynamoDBFully managed NoSQL database service for key-value and document storage.
12Amazon RedshiftData warehouse solution for analytical processing and large-scale datasets.
13Amazon VPCVirtual private cloud to launch AWS resources in a logically isolated environment.
14AWS IAMIdentity and Access Management to securely manage AWS permissions.
15AWS Auto ScalingAutomatically adjusts compute capacity to maintain steady performance.
16AWS CloudFormationInfrastructure as Code (IaC) for automating AWS resource provisioning.
17AWS CloudWatchMonitoring and observability service for AWS resources.
18AWS CodeDeployAutomates software deployments to various AWS environments.
19AWS CodePipelineCI/CD service for automating software release workflows.
20AWS GlueETL (Extract, Transform, Load) service for data preparation and integration.
21AWS Step FunctionsWorkflow automation to orchestrate AWS services into serverless applications.
22AWS KMSKey Management Service for encryption and secure key storage.
23Amazon SNSSimple Notification Service for messaging and event-driven notifications.
24Amazon SQSSimple Queue Service for message queuing between distributed components.
25AWS IoT CoreManaged cloud service for IoT device communication and management.
26AWS WAFWeb Application Firewall to protect against security threats.
27Amazon EKSManaged Kubernetes service for running containerized applications.
28Amazon ECSElastic Container Service for managing Docker containers.
29Amazon Elastic BeanstalkPlatform as a Service (PaaS) for deploying and scaling applications.
30AWS Security HubProvides centralized security compliance and threat monitoring.
31AWS Cost ExplorerTool for analyzing AWS cost usage and optimizing spending.
